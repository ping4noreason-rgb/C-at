# C-at
Collaborative C Editor

C-at is a specialized C code editor. It's lightweight and requires no more than 200-300 megabytes of RAM. The project is a one-person project. The hotkeys are:
1. Ctrl + Space (Disable tooltips)
2. Ctrl + Shift + Enter (Change panel background)
3. Ctrl + S (Save current file)
4. Ctrl + R (Register/Re-register with the server)
